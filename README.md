# 🚀 Focus Forge – Productivity & Career Tracking Platform

🔗 Live Demo: https://focus-forge-dev.vercel.app/

---

## 📌 Overview

Focus Forge is a centralized productivity platform designed to track and manage multiple aspects of a developer’s workflow in one place. It helps users stay consistent by organizing DSA practice, system design learning, job applications, and content creation.

---

## ✨ Features

- 📊 Track DSA and System Design progress  
- 🗂 Manage tasks and daily workflows  
- 📝 Draft and organize LinkedIn content  
- 📅 Calendar-based planning  
- 📈 Real-time progress analytics  
- 🔄 Multi-module productivity tracking  

---

## 🛠 Tech Stack

- Frontend: React.js, Tailwind CSS  
- Language: JavaScript  
- Deployment: Vercel  

---

## 🎯 Purpose

Built to reduce context-switching between multiple tools and create a structured system for consistent productivity and career tracking.
